# StoryHelper 랜딩 페이지 프로젝트 기획서

## 📋 프로젝트 개요

### 프로젝트 목적
StoryHelper Chrome 확장프로그램을 소개하고 홍보하기 위한 공식 랜딩 페이지 개발

### 타겟 사용자
- 티스토리 블로그 운영자
- SEO 최적화에 관심있는 블로거
- 생산성 향상 도구를 찾는 콘텐츠 크리에이터

---

## 🎯 StoryHelper 확장프로그램 분석

### 기본 정보
- **이름**: StoryHelper - 최고의 티스토리 생산성 확장프로그램
- **개발자**: shipfriend.dev
- **평점**: 5.0/5.0 (리뷰 2개)
- **사용자 수**: 169명
- **버전**: 1.6.0 (2025년 8월 16일 업데이트)
- **크기**: 93.84KiB

### 핵심 가치 제안
> "티스토리 블로그 운영의 생산성 향상을 경험하세요"

### 주요 기능

#### 1. 단축키 기능
- 자주 사용하는 기능을 단축키로 실행
- 사용자 맞춤 단축키 커스터마이징 지원

#### 2. SEO 이미지 최적화 도구
- 2가지 이미지 최적화 도구 제공
- SEO 성능 향상을 위한 이미지 처리

#### 3. 실시간 글자 수 카운팅
- 글 작성 중 실시간 글자 수 확인 UI
- 콘텐츠 최적화를 위한 즉각적인 피드백

#### 4. SEO 검증 기능 (v1.5 추가)
- 제목1(H1) 개수 검증: 글에 하나만 존재해야 함
- 이미지 대체 텍스트 검증: 모든 이미지에 alt 텍스트 필수
- 티스토리 SEO 규칙 준수 자동 체크

### 사용 방법
- 확장프로그램 클릭 시 나오는 화면에서 기능 활성화/비활성화 가능
- 기능 변경 후 새로고침 필요
- 단축키 커스터마이징 시 브라우저 기본 단축키와 충돌 주의

### 개인정보 보호
- 데이터 수집 없음
- 서드파티 판매 없음
- 핵심 기능 외 목적으로 데이터 사용/전송 없음

## 🎨 랜딩 페이지 구성 제안

### 1. Hero Section (히어로 섹션)
**목표**: 첫 인상에서 명확한 가치 전달

**콘텐츠**:
- 메인 헤드라인: "티스토리 블로그 생산성을 10배 높이는 확장프로그램"
- 서브 헤드라인: "SEO 최적화부터 글자 수 카운팅까지, StoryHelper와 함께라면 간단합니다"
- 주요 CTA: "Chrome에 추가하기" (Chrome Web Store 링크)
- 보조 CTA: "기능 살펴보기" (스크롤 다운)
- 히어로 이미지/애니메이션: 확장프로그램 사용 화면 또는 인터랙티브 데모

**시각 요소**:
- 확장프로그램 아이콘 및 브랜딩
- 사용자 수 배지 (169+ 사용자)
- 평점 표시 (⭐ 5.0/5.0)

### 2. Problem & Solution Section (문제 및 해결책)
**목표**: 사용자 페인 포인트 공감 및 해결 방법 제시

**블로거가 겪는 문제들**:
- ❌ SEO 규칙을 지키기 어려움 (H1 중복, 이미지 alt 누락)
- ❌ 이미지 최적화에 시간 소요
- ❌ 글자 수를 수동으로 확인해야 함
- ❌ 반복 작업이 많아 비효율적

**StoryHelper의 해결책**:
- ✅ 자동 SEO 검증으로 실수 방지
- ✅ 원클릭 이미지 최적화
- ✅ 실시간 글자 수 표시
- ✅ 단축키로 빠른 작업 처리

### 3. Features Section (기능 소개)
